WS_FTPの設定ファイル( WS_FTP.INI）から、ホストの設定をインポートできます。

接続メニューのインポート−WS_FTPを選びます。ファイルの選択画面で、WS_FTP.INIを選びます。

anonymous-FTP以外のパスワードはインポートしません。インポートした後パスワードを設定してください。anonymous-FTPの場合はメールアドレスをパスワードとして使用します。


WS_FTP.INIファイルは、以下のような内容です。

[Host Name]
HOST=xxx.yyy.com
UID=sota
LOCDIR=c:\download
DIR=/pub
PASVMODE=1                         （値は 0 または 1）
FIREWALL=1                         （値は 0 または 1）

[Host Name 2]
HOST=xxx.yyy.zzz.com

　　：
